## Yii2 module tools

Модуль реализует вспомогательные утилиты.

Реализованы следующие контроллеры:
* [db](./docs/controllers/db.md)

Установка:

### Composer

```console
composer require devskyfly/yii-module-yii-tools 
```
### Yii2

```php
'yii-tools'=>[
	'class'=>'devskyfly\yiiModuleTools\Module',
],
```
